# Student CRUD Application

A simple **PHP & MySQL** web application to manage student records.  
This project demonstrates **CRUD operations** (Create, Read, Update, Delete) using **PHP**, **MySQL**, and **Bootstrap 5**.

---

## Features

- Add new student records
- View all students
- Edit existing student records
- Delete student records
- Clean and responsive Bootstrap UI

---

## Technology Stack

- PHP (Vanilla)
- MySQL / MariaDB
- Bootstrap 5
- XAMPP / Local PHP server
